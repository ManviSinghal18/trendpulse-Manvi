# 🔥 TrendPulse Project

## 📌 Overview

TrendPulse is a Python-based data analysis project that collects trending stories and identifies the most popular topics using text processing and visualization.

---

## 🚀 Tasks Completed

### ✅ Task 1: Data Collection

* Collected 100+ trending stories
* Stored the data in JSON format (`trends_20260410.json`)

---

### ✅ Task 2: Trend Analysis

* Cleaned text data using regex
* Removed stopwords (common unnecessary words)
* Extracted most frequent (trending) words
* Saved results in `top_trends.json`

---

### ✅ Task 3: Data Visualization

* Created a bar chart using Matplotlib
* Displayed top trending words
* Saved graph as `trend_graph.png`

---
